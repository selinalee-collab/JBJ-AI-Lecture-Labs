# M10 · API

## 실습 목표

API 키, 요청, 응답의 흐름을 이해하고 Open DART에서 최근 공시를 조회합니다.

## 준비

1. [Open DART](https://opendart.fss.or.kr)에서 인증키를 발급받습니다.
2. 프로젝트의 `.env`에 키를 저장합니다.
3. `.env`를 `.gitignore`에 추가합니다.

```dotenv
DART_API_KEY=발급받은_키
```

```gitignore
.env
```

실제 키를 README, 코드, 프롬프트, Git 커밋에 입력하지 않습니다.

## 실습

```text
.env의 DART_API_KEY를 환경변수로 읽어 Open DART API에서 삼성전자의 최근 공시 10건을 가져오는 프로그램을 만들어줘.
접수일, 보고서명, 제출인을 표로 저장하고 인증 실패, 조회 결과 없음, 네트워크 오류를 구분해 처리해줘.
API 키는 코드나 로그에 출력하지 마.
```

## 완료 확인

- [ ] API 키를 환경변수로 분리했다.
- [ ] `.env`가 Git 추적 대상에서 제외됐다.
- [ ] 최근 공시 10건을 표로 확인했다.
- [ ] 오류 처리와 로그를 확인했다.

# M07 · Skill

## 실습 목표

공개 Skill을 검토·설치하고, 영수증 이미지에서 정보를 읽어 회계장부를 만드는 Skill을 제작합니다.

## 제공 파일

| 파일 | 용도 |
|---|---|
| [`회계장부_실습.xlsx`](./회계장부_실습.xlsx) | 영수증입력·회계장부·카테고리별집계 3시트 양식 |
| [`스킬_학습용.zip`](./스킬_학습용.zip) | Skill 제작용 영수증 10장 |
| [`스킬_테스트용.zip`](./스킬_테스트용.zip) | 완성 Skill 검증용 영수증 10장 |

## 실습 1 · Skill 설치

1. [skills.sh](https://skills.sh)에서 `spreadsheet`를 검색합니다.
2. 설치 수, 작성자, `SKILL.md` 내용을 확인한 뒤 하나를 설치합니다.
3. 에이전트를 다시 시작하고 사용 가능한 Skill 목록을 확인합니다.
4. 간단한 엑셀 파일 생성을 요청해 동작을 검증합니다.

## 실습 2 · 영수증 자동 정리 Skill 제작

1. `스킬_학습용.zip`을 별도 작업 폴더에 풉니다.
2. 회계장부 양식을 같은 폴더에 복사합니다.
3. 에이전트에게 다음과 같이 요청합니다.

```text
이 영수증 이미지를 날짜, 가맹점, 품목, 금액으로 읽어 식비·교통비·도서비·사무용품·기타로 분류해줘.
반복 실행할 수 있는 Skill을 만들고, 원본은 보존하면서 정리완료/<카테고리>/에 정돈된 사본을 저장해줘.
회계장부_실습.xlsx 양식을 이용해 결과 엑셀도 새 파일로 만들어줘.
```

4. `스킬_테스트용.zip`으로 처음 보는 영수증도 처리되는지 확인합니다.

## 완료 확인

- [ ] 외부 Skill의 출처와 내용을 검토하고 설치했다.
- [ ] 직접 만든 Skill에 `SKILL.md`가 있다.
- [ ] 카테고리별 영수증 사본과 결과 엑셀을 생성했다.
- [ ] 테스트용 자료로 재실행했다.

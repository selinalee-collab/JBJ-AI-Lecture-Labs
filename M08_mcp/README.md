# M08 · MCP

## 실습 목표

MCP의 클라이언트·서버 구조를 이해하고 Notion과 PlayMCP 도구를 연결합니다.

## 실습 1 · 연결 상태 확인

1. 사용하는 에이전트의 MCP 설정에서 Notion을 연결하고 인증합니다.
2. 세션의 `/mcp` 또는 CLI의 MCP 목록 명령으로 연결 상태를 확인합니다.
3. 접근 가능한 Notion 페이지를 검색하고 제목만 읽어봅니다.

## 실습 2 · PlayMCP 재고 조회

1. [PlayMCP](https://playmcp.kakao.com)에 로그인합니다.
2. 도구함에 리테일 재고 조회 도구를 추가하고 에이전트에 연결합니다.
3. 연결 확인 후 다음 요청을 실행합니다.

```text
PlayMCP의 리테일 도구를 사용해 현재 위치 주변 매장을 찾고, 내가 지정한 상품의 재고를 확인해줘. 매장명, 거리, 재고 상태를 표로 정리해줘.
```

서비스의 도구명과 제공 범위는 변경될 수 있으므로 실습 당일 PlayMCP 화면을 기준으로 선택합니다.

## 완료 확인

- [ ] MCP 연결 상태를 확인했다.
- [ ] Notion에서 읽기 작업을 수행했다.
- [ ] 리테일 도구가 매장 검색과 재고 조회를 순서대로 수행했다.

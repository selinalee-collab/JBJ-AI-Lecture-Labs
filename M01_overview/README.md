# M01 · 개관 · 도구 소개

## 실습 목표

- 일반 생성형 AI와 AI 에이전트의 차이를 설명합니다.
- Claude Code와 Codex 중 수업에서 사용할 도구를 정합니다.
- 에이전트가 파일과 명령을 다룬다는 점을 이해합니다.

## 실습

1. 일반 챗봇에 "폴더의 엑셀 파일을 합쳐 표를 만들어줘"라고 요청했을 때 필요한 수작업을 적습니다.
2. AI 에이전트가 대신할 수 있는 단계인 파일 탐색, 코드 작성, 명령 실행, 결과 저장을 구분합니다.
3. 본인의 반복 업무 한 가지를 골라 아래 형식으로 작성합니다.

```text
입력 파일:
반복 작업:
원하는 결과:
검토할 기준:
```

## 완료 확인

- [ ] 챗봇과 에이전트의 차이를 한 문장으로 설명했다.
- [ ] 사용할 에이전트 도구를 정했다.
- [ ] 자동화할 반복 업무를 하나 정의했다.

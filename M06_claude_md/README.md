# M06 · CLAUDE.md 심화

## 실습 목표

프로젝트에서 반복할 설명을 `CLAUDE.md` 또는 `AGENTS.md`에 상시 지침으로 작성하고 적용 여부를 확인합니다.

## 시작 양식

```markdown
# 프로젝트 지침

## 응답 방식
- 한국어로 간결하게 답한다.
- 작업 전에 변경할 파일을 먼저 설명한다.

## 파일 규칙
- 원본 엑셀 파일은 덮어쓰지 않고 사본에 작업한다.
- 결과 파일명에는 `_result`를 붙인다.

## 완료 조건
- 변경 후 관련 파일과 실행 결과를 확인한다.
```

## 실습 순서

1. 프로젝트 루트에 Claude Code는 `CLAUDE.md`, Codex는 `AGENTS.md`를 만듭니다.
2. 응답 호칭이나 형식처럼 바로 확인할 수 있는 규칙을 하나 추가합니다.
3. 새 세션에서 간단한 파일 작업을 요청하고 규칙 적용 여부를 확인합니다.
4. 선택 실습으로 하위 폴더에 더 구체적인 지침 파일을 두고 우선순위를 비교합니다.

## 완료 확인

- [ ] 루트 지침 파일을 작성했다.
- [ ] 응답 규칙과 파일 보호 규칙을 포함했다.
- [ ] 새 세션에서 지침 적용을 확인했다.

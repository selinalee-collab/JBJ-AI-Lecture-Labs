# M09 · Sub-agent

## 실습 목표

서로 다른 검토 기준을 Sub-agent에 나누어 맡기고 결과를 하나의 보고서로 통합합니다.

## 준비

[M07의 `회계장부_실습.xlsx`](../M07_skill/회계장부_실습.xlsx) 또는 M07에서 만든 결과 장부를 작업 폴더에 복사합니다.

## 실습

```text
회계장부에 대해 월말 경비 결산 점검을 수행해줘.

1. 증빙 검토 에이전트: 날짜·가맹점·금액·증빙 누락 확인
2. 분류 검토 에이전트: 계정과목·카테고리 적정성 확인
3. 이상치 검토 에이전트: 중복·고액·비정상 패턴 확인

세 Sub-agent는 독립적으로 검토하고, 메인 에이전트는 결과를 중복 제거해 중요도 순으로 정리한 1페이지 Markdown 보고서를 만들어줘. 원본 엑셀은 수정하지 마.
```

먼저 한 에이전트에게 전체 점검을 시킨 결과와 비교해 누락과 근거의 차이를 확인합니다.

## 완료 확인

- [ ] 역할이 다른 Sub-agent 세 개를 실행했다.
- [ ] 각 결과에 근거가 포함됐다.
- [ ] 메인 에이전트가 중복을 정리한 보고서를 만들었다.
- [ ] 원본 엑셀을 보존했다.

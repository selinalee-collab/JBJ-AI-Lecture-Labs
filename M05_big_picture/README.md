# M05 · 전체 구조 조망

## 실습 목표

지침, Skill, MCP, Sub-agent의 역할과 프로젝트 안에서의 위치를 구분합니다.

## 실습

연습 프로젝트에서 다음 구조를 직접 만들거나 에이전트에게 생성을 요청합니다.

```text
my-project/
├── CLAUDE.md 또는 AGENTS.md
├── .claude/
│   ├── skills/
│   └── agents/
└── .codex/
    ├── skills/
    └── agents/
```

```text
이 프로젝트에 Claude Code와 Codex의 프로젝트 지침, Skill, Sub-agent 폴더 구조를 만들어줘. 빈 폴더가 Git에 보존되도록 처리하고 각 위치의 역할을 설명해줘.
```

## 완료 확인

- [ ] 네 구성요소의 역할을 한 줄씩 설명했다.
- [ ] 글로벌 설정과 프로젝트 설정을 구분했다.
- [ ] 프로젝트 폴더 구조를 직접 확인했다.
